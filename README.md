# SPHpy:
Smoothed Particle Hydrodynamics written in python.

---
## Requirements

- python 2.7.x
- pyOpenGL

---
## Keybind

- k: rotate simulation box
- j: rotate simulation box
- J: box size --
- K: box size ++
- t: dt--
- T: dt++
- r: shuffle particles
- e: wall(e) --
- E: wall(e) ++
- v: viscosity ++
- V: viscosity --
- [Space]: increase particles
- -: reduce particles
- 1: for debug
- 2: change 3D->2D
- q: quit
- h: show this message
